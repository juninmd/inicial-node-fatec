# Inicial Node Fatec

**Description:** A standard software project.

**Installation:**

1.  Clone the repository: `git clone https://github.com/inicial-node-fatec.git`
2.  Navigate to the project directory: `cd inicial-node-fatec`
3.  Install dependencies: `npm install`
4.  Configure the server: `npm run start`
5.  Run the server: `npm run dev`

**Usage:**

1.  **Understanding the Files:**
    *   `.gitignore`: Specifies files to be ignored by Git.
    *   `app`: The main application logic.
    *   `config`: Configuration settings for the application.
    *   `package.json`: Defines project metadata, dependencies, and scripts.
    *   `public`: Contains static assets (e.g., images, CSS, JavaScript).
    *   `renovate.json`:  Used for code versioning and automated testing.
    *   `server.js`:  The main server file, responsible for handling requests.
    *   `yarn.lock`:  A lock file ensuring consistent project state across environments.

2.  **Running the Application:**
    *   Start the server: `npm start`
    *   Access the application in your browser at `http://localhost:3000`

3.  **Development:**
    *   Navigate to the `app` directory: `cd app`
    *   Run the development server: `npm run dev`
    *   Inspect the `public` directory for assets.
    *   Run the server via `npm run dev` again.
